Giá trị X bằng:

Vấn đề

Tìm x trong phương tình ax2 + bx + c = 0

Bài toán

Giải phương trình bậc hai: Viết chương trình sử dụng cấu trúc điều kiện để giải phương trình bậc hai có dạng ax^2 + bx + c = 0 với các hệ số a, b, c là các số nguyên đã cho. Nếu phương trình có nghiệm, in ra các nghiệm của nó
